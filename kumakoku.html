    漢字の筆順プログラム

    漢字の書き順プログラムをご紹介いたしました。今回は、「ある漢字を初見で学ぶ」という疑似体験を先生方にしていただくために、一般的に馴染みのない「サンスクリット語」の「あ（読み）」をプログラミングで学んでいただきました。

    書き順プログラム（サンスクリット語：「あ」）

    sample program 練習　program

    サンスクリット語：「あ」の筆順プログラム 全メガネ

    漢字の組み合わせプログラム

    漢字の組み合わせプログラムをご紹介いたしました。十画以上の漢字の場合、前述の筆順プログラムは作成の手間が多くなり、プログラム作成の時間がかかり過ぎてしまいます。今回ご紹介した「ビャンビャン麺」の「ビャン」という漢字は54画。このように画数が多が多い漢字の場合、既存の漢字の組み合わせであることが多いです。その組み合わせをプログラムして漢字を表現する中で、え「エピソード記憶」を利用して学習効果を狙うものです。

    漢字の組み合わせプログラム（「ビャンビャン麺の「ビャン」）

    sample program 練習　program
